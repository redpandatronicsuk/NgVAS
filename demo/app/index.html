NgVAS


Angular directive for a Visual Analog Scale (VAS)

Description

NgVAS is a Visual Analog Scale (VAS) Angular directive. VAS can be used to collect responses from users on a continous scale. HCI research has shown that the VAS have superior accuracy than using sliders as the VAS is not being biased by having a starting postion [Funke 2015].

There have also been studies comparing continous scales (e.g. sliders and VAS) and categorical scales, such as radio boxes [Cook 2001]. VAS are widely used by medical reserchers to get pain feedbacks from their patients [google scholar].

Parameters

width
the width of the VAS (default: 100%)
height
the height of the VAS (default: 40px)
line-height
the height of the clickable/draggable horizontal line in center of the VAS. NOTE: decimal values can be used, e.g 0.25 (default: 3)
marker-type
the marker type used to display the selected value, valid values: cross, line, point, arrow-up, arrow-down, arrow-left, arrow-right (default: cross)
marker-color
the color of the marker. NOTE: this is for convinience, the color can also be set with css: .vas-mark {stroke: ...} (default: blue)
min
the value on the left extrema of the VAS (default: 0)
max
the value on the right extrema of the VAS (default: 100)
cm-length
variable to store the click length converted to centimeters in. This may be useful to correct for biases caused by different screen sizes. NOTE: This is only an approximation. We looked for ways to convert pixels to centimeter values (http://stackoverflow.com/questions/1341930/pixel-to-centimeter, http://stackoverflow.com/questions/279749/detecting-the-system-dpi-ppi-from-js-css, ...) and the best we found was this: http://stackoverflow.com/questions/10794891/getting-height-of-a-div-in-centimeter-using-jquery. We measured the actual length in centimeters with a ruler and they didn't completely match. However, we also measured a box like <div style="background:black;heigth:1cm;width:10cm;"></div>; and it is not 10cm wide when measured with a ruler. Please leave a comment below if you know a better solution!

Example

Simple

To use NgVAS with the default parameters simply include the ng-vas tag in your HTML, where you want the VAS to appear: 

<ng-vas ng-model="vas1"></ng-vas> {{vas1 | number:2}}

{{vas1 | number:2}}

By default the scale will be on a range from 0 to 100, the only parameter that needs to be specified is ng-model. The default width is 100%, so it will span the entire width of the parent container. Width and other parametes can be specified, as can is shown in the next section.

Attributes and Styling

You can set the height and width as attributes: 

<ng-vas ng-model="vas2" width="200px" height="50px"></ng-vas>

NgVAS was designed for data collection only. As labels on the sides of the scale and displaying feedback (such as the numerical value of the marker postion), can bias the user, we have ommitted these in NgVAS, but if you would like such functionallity you can easily implement it yourself in HTML, such as in this example:
